# AI Resume Screening System

## 📌 Objective
This project is an AI-based Resume Screening System that evaluates candidates based on a given Job Description.

## 🚀 Features
- Skill Extraction from Resume
- Matching Resume with Job Description
- Candidate Scoring (0–100)
- Explanation of Score

## 🔄 Pipeline
Resume → Extract → Match → Score → Explain

## 🛠️ Technologies Used
- Python
- Basic NLP logic
- Google Colab

## 📂 Project Structure
- main.py → Main execution file
- chains/ → Logic for extraction, matching, scoring
- prompts/ → Prompt templates (for structure)
- final_results.txt → Output results

## 👨‍💻 Candidates Tested
- Strong Candidate
- Average Candidate
- Weak Candidate

## 📊 Output
The system provides:
- Extracted Skills
- Matched Skills
- Missing Skills
- Final Fit Score (out of 100)
- Explanation of the score

## ⚠️ Note
Due to lightweight model limitations, a clean Python logic-based pipeline is used for stable and explainable output.

## 📌 Conclusion
This project demonstrates how AI can be used to automate resume screening and assist recruiters in candidate evaluation.
